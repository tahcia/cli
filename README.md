# Tahcia CLI

Tahcia CLI For Mac &amp; Linux
Tahcia CLI is a local automation runtime for browser and desktop workflows.

It allows scripts, AI agents, and MCP clients to automate:

- Chrome
- macOS desktop
- keyboard/mouse actions
- file workflows
- browser tabs
- downloads/uploads
- enterprise web tools

Tahcia combines browser automation with native desktop control, allowing workflows that traditional browser automation tools cannot reliably handle.

##
This CLI repo doesn't come with Tahcia Google Chrome extension. 
[Please download it here.](https://www.github.com/tahcia/chrome)
